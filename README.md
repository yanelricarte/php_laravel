# Blog Laravel

Proyecto inicial desarrollado con Laravel para practicar rutas, controladores, vistas Blade y publicación en GitHub.

## Tecnologías utilizadas

- PHP 8
- Laravel
- Composer
- Blade
- Git
- GitHub

## Funcionalidades

- Página de inicio
- Página de materias
- Página de contacto
- Layout base reutilizable
- Navegación entre secciones
- Datos enviados desde el controlador a las vistas
